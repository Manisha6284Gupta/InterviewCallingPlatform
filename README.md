🚀 Codify – Modern Developer Platform

Codify is a modern developer platform for coding, collaboration, and learning.
It provides tools for real-time communication, code editing, authentication, and developer collaboration.
The project is built using a modern full-stack architecture with React, Node.js, Express, and modern developer tools.

---

🌟 Features

🔐 Secure Authentication (Clerk)

💻 Built-in Code Editor (Monaco Editor)

💬 Real-time Chat System

📹 Video Communication

⚡ Fast Development Environment with Vite

🎨 Modern UI with TailwindCSS + DaisyUI

🔔 Toast Notifications

📊 Efficient Data Fetching using React Query

🎉 Interactive UI Animations

📡 REST API backend

Frontend Setup

Clone the repository

git clone https://github.com/Manisha6284Gupta/InterviewCallingPlatform

Navigate to frontend folder

cd codify/frontend

Install dependencies

npm install

Run development server

npm run dev

Build for production

npm run build

Preview production build

npm run preview

🔑 Environment Variables

Create a .env file inside both frontend and backend directories.

Example:

# Clerk
CLERK_SECRET_KEY=
VITE_CLERK_PUBLISHABLE_KEY=

# Stream
STREAM_API_KEY=
STREAM_SECRET=

# Backend
PORT=5000
MONGO_URI=
